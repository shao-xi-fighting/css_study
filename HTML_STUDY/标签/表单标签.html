type属性的属性值描述

属性值 描述
button 定义可点击按钮
checkbox定义复选框
file定义输入字段和浏览按钮，供文件上传
hidden 定义隐藏的输入字段
image 定义图像提交按钮
password 定义密码字段，该字段字符被掩码
radio 定义单选按钮
reset 定义重置按钮。重置按钮会清除表单所有数据
submit 定义提交按钮，表单数据被提交
text 定义单行文本输入默认宽度为20字符

input表单元素

属性值 属性值描述
name 用户自定义 定于input元素名称
value 用户自定义规定input元素的值
checked checked 规定input首次加载显示的值
maxlength 正整数 规定输入字符段字长
